# リッチメニュー設定手順書（LINE Official Account Manager）

こめもっち LINE公式アカウントのリッチメニュー（`assets/rich-menu.png`）を設定する手順です。

## 画像の仕様

| 項目 | 値 |
|------|-----|
| ファイル | `assets/rich-menu.png` |
| サイズ | 2500 × 1686 px（大サイズ） |
| 分割 | 6分割（2行 × 3列） |
| ファイル形式 | PNG（1MB 以下） |

画像を修正したい場合は `assets/rich-menu.svg` を編集し、`npm run generate:rich-menu` で PNG を再生成してください（初回実行時に Zen Maru Gothic フォントを自動ダウンロードします）。

## 1. アップロード手順

1. [LINE Official Account Manager](https://manager.line.biz/) にログイン
2. 対象アカウントを選択 → 左メニュー **トークルーム管理 > リッチメニュー**
3. **作成** をクリック
4. 表示設定を入力
   - タイトル: 「こめもっちメニュー」（管理用の名前。ユーザーには表示されない）
   - 表示期間: 開始日を本日、終了日は未設定（無期限）
   - メニューバーのテキスト: 「メニュー」
   - メニューのデフォルト表示: **オン**（トークを開いたときに自動で開く）
5. **コンテンツ設定 > テンプレートを選択** で **大 > 6分割（2行×3列）** を選択
   - 左上から Z 字順に A〜F のエリアが割り当てられるテンプレートです
6. **画像を作成** ではなく **背景画像をアップロード** で `assets/rich-menu.png` を選択
7. 下記「アクション設定」のとおり A〜F のアクションを入力
8. **保存** → プレビューで見た目とタップ領域を確認して公開

## 2. 6分割エリアのタップ座標

テンプレート「大・6分割」を使えば座標は自動設定されますが、Messaging API で
リッチメニューを作成する場合（将来の自動化用）は以下の bounds を使用します。

| エリア | メニュー | x | y | width | height |
|--------|---------|-----|-----|-------|--------|
| A（左上） | スタンプカード | 0 | 0 | 833 | 843 |
| B（中央上） | こめもっちストア | 833 | 0 | 834 | 843 |
| C（右上） | 新商品情報 | 1667 | 0 | 833 | 843 |
| D（左下） | お問い合わせ | 0 | 843 | 833 | 843 |
| E（中央下） | お客様の声 | 833 | 843 | 834 | 843 |
| F（右下） | マイページ | 1667 | 843 | 833 | 843 |

※ 3列の合計が 2500px になるよう中央列のみ 834px です。

## 3. 各エリアのアクション設定

| エリア | メニュー | アクションタイプ | 設定値 |
|--------|---------|----------------|--------|
| A | スタンプカード | リンク | `https://liff.line.me/{LIFF_ID}`（LIFF登録後のURL。LIFF IDは LINE Developers Console で確認） |
| B | こめもっちストア | リンク | Shopifyストアのトップ `https://{SHOPIFY_SHOP_DOMAIN}/` |
| C | 新商品情報 | リンク | 新商品コレクション `https://{SHOPIFY_SHOP_DOMAIN}/collections/new`（または新着案内ページ） |
| D | お問い合わせ | リンク | お問い合わせフォーム `https://{SHOPIFY_SHOP_DOMAIN}/pages/contact`（テキスト自動応答にする場合は「テキスト」で `お問い合わせ` を送信させ、応答メッセージを設定） |
| E | お客様の声 | リンク | レビューページ（楽天レビュー or Shopifyのレビューページ URL） |
| F | マイページ | リンク | Shopifyアカウントページ `https://{SHOPIFY_SHOP_DOMAIN}/account` |

- `{LIFF_ID}` / `{SHOPIFY_SHOP_DOMAIN}` は `.env.local` に設定した実際の値に置き換えてください
- リンクのアクションには「アクションラベル」の入力が必要です（音声読み上げ用。各メニュー名をそのまま入れればOK）
